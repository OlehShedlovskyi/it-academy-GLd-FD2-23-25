Задание по получению списка задач (ToDo) из сервиса JSONPlaceholder и созданию интерфейса для перетаскивания задач (Drag and Drop) в браузере:


Шаг 1: Получение данных из JSONPlaceholder
Запрос данных: Используйте метод fetch для получения данных о задачах (ToDo) из API JSONPlaceholder. URL для получения списка задач: https://jsonplaceholder.typicode.com.
Обработка данных: После получения данных, вам нужно сохранить их в переменной или состоянии для дальнейшего использования. Количество задач можно ограничить, например 20
Шаг 2: Создание интерфейса задач
HTML структура: Создайте базовую структуру HTML, в которой будут отображаться задачи.
Шаг 3: Реализация Drag and Drop
Добавление обработчиков событий: Добавьте обработчики событий для перетаскивания (drag) и отпускания (drop) задач.
