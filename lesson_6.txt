Задача: Управление транспортными средствами
Создайте базовый класс Vehicle, который будет представлять транспортное средство, и расширьте его для создания классов Car и Truck, которые будут представлять конкретные типы транспортных средств.
Шаги:
Создайте класс Vehicle с следующими свойствами:
make (марка)
model (модель)
year (год выпуска)
mileage (пробег)
И методами:
drive(distance): увеличивает пробег на указанное расстояние
info(): возвращает строку с информацией о транспортном средстве
Создайте класс Car, который расширяет класс Vehicle, с дополнительным свойством:
numDoors (количество дверей)
И методами:
info(): переопределяет метод info() базового класса, добавляя информацию о количестве дверей
Создайте класс Truck, который расширяет класс Vehicle, с дополнительным свойством:
cargoCapacity (грузоподъемность)
И методами:
info(): переопределяет метод info() базового класса, добавляя информацию о грузоподъемности
Пример использования:
javascript
// Пример использования
const myCar = new Car('Toyota', 'Corolla', 2015, 50000, 4);
console.log(myCar.info()); // "2015 Toyota Corolla with 50000 miles and 4 doors"
myCar.drive(150);
console.log(myCar.info()); // "2015 Toyota Corolla with 50150 miles and 4 doors"


const myTruck = new Truck('Ford', 'F-150', 2018, 30000, 1000);
console.log(myTruck.info()); // "2018 Ford F-150 with 30000 miles and 1000 kg cargo capacity"
myTruck.drive(200);
console.log(myTruck.info()); // "2018 Ford F-150 with 30200 miles and 1000 kg cargo capacity"




Дополнительные задания:
Добавьте в класс Vehicle метод service(), который будет устанавливать пробег на 0 и выводить сообщение о том, что транспортное средство прошло обслуживание.
Добавьте в класс Car метод openTrunk(), который будет выводить сообщение о том, что багажник открыт.
Добавьте в класс Truck метод loadCargo(weight), который будет проверять, можно ли загрузить указанный вес, и выводить соответствующее сообщение.
